# Odusanya Uthman - Portfolio Website

A modern, responsive portfolio website showcasing frontend development skills with interactive features and seasonal animations.

## 🚀 Features

### Core Features
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Interactive UI** - Smooth animations, hover effects, and transitions
- **Collapsible Sections** - Experience and Skills sections with "View More" functionality
- **Contact Form** - EmailJS integration for form submissions
- **Project Showcase** - 11+ featured projects with live links and modal previews
- **Keyboard Navigation** - Full accessibility support with skip links and focus management

### Special Features
- **Shining Logo Animation** - Star-like glow effect on "Odusanya Uthman" text
- **Typewriter Effect** - Animated text in hero section
- **Back to Top** - Contextual floating button that appears when scrolling
- **Image Modal** - Click project images for enlarged view
- **Seasonal Effects** - Snow in December, fireworks January 1-3

### Technical Features
- **Progressive Enhancement** - Works without JavaScript
- **Performance Optimized** - Lazy loading, compressed assets
- **SEO Friendly** - Meta tags and semantic HTML
- **Cross-browser Compatible** - Modern browser support

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic markup and accessibility
- **CSS3** - Modern styling with animations and responsive design
- **JavaScript (ES6+)** - Interactive functionality and DOM manipulation

### Libraries & Tools
- **EmailJS** - Contact form backend service
- **Google Fonts** - Montserrat and Inter typefaces
- **Responsive Images** - Optimized loading with lazy attributes

### Development Tools
- **Visual Studio Code** - Code editor
- **Git** - Version control
- **Netlify/Vercel** - Hosting and deployment

## 🚀 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd portfolio
   ```

2. **Open in browser**
   - Open `index.html` in your web browser
   - Or use a local server for better performance

3. **EmailJS Setup** (for contact form)
   - Sign up at [EmailJS](https://www.emailjs.com/)
   - Create a service and template
   - Update credentials in `script.js`

4. **Resume Upload**
   - Place your resume PDF as `images/Uthman_Odusanya_Resume.pdf`
   - Update download link if needed
